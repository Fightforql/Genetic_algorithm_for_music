### 安装步骤

1. 克隆代码库：
   ```bash
   git clone https://github.com/username/project-name.git
    ```
2. 配置Fluidsynth：
   从https://github.com/FluidSynth/fluidsynth/releases 中下载，并将含有fluidsynth.exe的目录添加到环境变量
3. 进入项目目录，安装依赖项：
  ```bash
   pip install -r requirements.txt
   ```
4. 运行项目
   ```bash
   python -m genetic.main
   ```
